
### Shell environment set up for building Embi OS image. ###

You can now run 'BB_NUMBER_THREADS="6" bitbake <target>'

Common targets are:
    b2qt-embedded-qt6-image
    meta-toolchain-b2qt-embedded-qt6-sdk

QBSP target is:
    meta-b2qt-embedded-qbsp
    
If you only want to fetch source before running the build, simply add '--runall=fetch'

For more information about Embi OS, see https://github.com/Embi-OS/
